# Coconut-Game webpage document

## How to start
Run the command
```source init.sh```
and the service will be on

### For general users
## Voting Stage

- Click the grid to place some ballot(s)
- Long-click the grid to remove the ballot(s)

## Final Stage
- Drag the cluster onto the grids on your turn

### For admin
- Reveal results
- Start the final phase
