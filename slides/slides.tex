\begin{frame}{島嶼與棲地} 
  \vspace{0.4em}
  \large
  \textbf{島嶼}最初為一個由六角塊組合而成的地圖，一個六角塊則相當於島上的一塊\textbf{棲地}。
\end{frame}

% page 
\begin{frame}{猴子}
  \vspace{0.4em}
    \large
    \text{猴子}是島上的居民，居住於棲地內。此外，每隻猴子都會有各自支持的小隊。
\end{frame}

% page 
\begin{frame}{棲地單調條款}
  \vspace{0.4em}
  \large
  某個階段後（後續將詳述），同一棲地內的猴子會轉而支持原先在該棲地內獲得多數支持的小隊，即在棲地內贏者全拿。
\end{frame}

% page 
\begin{frame}{蠑螈法案}
  \vspace{0.4em}
  \large
  最終多個相連的棲地可能被統一計算，在其中佔猴子支持度絕對多數的小隊可以獲得相等於這些棲地內猴子總數的票數。
\end{frame}

% page 
\begin{frame}{遊戲流程}
  \vspace{0.4em}
  \large
  遊戲共有兩個階段，第一個階段小隊要在島嶼的各個棲地上分配自己的支持者，第二階段則是將多個棲地劃到一起，一起的棲地最終將被統一計算支持度。
\end{frame}

% page 
\begin{frame}{階段一：拉幫結社}
  \vspace{0.4em}
  \begin{itemize}
    \item \textbf{目標：}小隊分配支持自己的猴子到各個棲地上
    \item 共進行三輪
    \item 每輪每隊最多可以分配 100 隻猴子
    \item 前兩輪結束時會公布所有小隊的分配情形
    \item 每一輪分配時間 3 分鐘，中途不暫停
  \end{itemize}
\end{frame}

% page 
\begin{frame}{階段一：拉幫結社}
  \vspace{0.4em}
  \begin{itemize}
    \item 第三輪結束時將結算猴子的支持狀況
    \item 每個棲地內有最多數量猴子的小隊將可以獲得該棲地內所有猴子
    \item 若不存在絕對多數，則棲地內所有猴子都會集體消失
  \end{itemize}
\end{frame}

% page 
\begin{frame}{階段二：搶奪地盤}
  \vspace{0.4em}
  \begin{itemize}
    \item 每個小隊有一些包含五個相連棲地的板塊
    \item 小隊依\textbf{階段一}放置猴子的變異數由大至小輪流放板塊
    \item 放置時限 30 秒
  \end{itemize}
  （圖待補）
\end{frame}

% page 
\begin{frame}{階段二：搶奪地盤}
  \vspace{0.4em}
  \begin{itemize}
    \item 板塊框住的棲地適用\textbf{蠑螈法案}
    \item 每組板塊內有最多猴子的小隊獲得相等於猴子總數的分數
  \end{itemize}
